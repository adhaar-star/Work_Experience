INSTALL & UPGRADE

INSTALLATION

Installing the script is very easy. Follow the steps below to install this script on a local machine or on your server. If you do not currently have a server.

  1. Move all the files inside of the ‘jarvis’ folder to the root of your website.
  2. Create a new database on your server and give it any name you would like to use.

  3. Navigate to ‘yourwebsite.com/jarvis-chat-folder/’ and you will be presented with an install screen. The install screen should look similar to the following image: 

5. On the install screen, you’ll want to add your Admin Username, Email, and Password. Next you’ll         enter your database info including your Database Host, Name, User, and Password.

6.Click install and if everything went through correctly, you should see a screen similar to the following: 

 

7. Then just ‘Click Here to Visit Your Site’ and you’ll be taken to your new site.

 

Note, if you are having trouble installing, don't hesitate to sent message to me to support you at php.power.arts@gmail.com.

Customize your own site.

UPGRADING

Follow the steps below to upgrade your script to the latest version:

  1. Make sure you have backed up all the files inside of the ‘application’ and ‘assets’ folder

  2. Upload and replace all files and folders in the application folder (EXCEPT CONFIG FOLDER) (note, if you have changed any of the language files, you do not need to replace the ‘lang’ folder and you can keep your current custom language settings)

  3. Upload and replace all the files inside of the ‘assets’ folder

  4. You can use the upgrade tools in your website to make this steps for you, just compress all files inside the script folder in zip file except (config,language,assets/upload) folder then select the zip file from setting at the bottom of the page then press upgrade and have fun :)

  5. Visit yoursite.com and see the updates

Test your site is backup and everything should be all good to go!
